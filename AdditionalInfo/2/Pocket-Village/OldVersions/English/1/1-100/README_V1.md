# GitHub organization: Pocket-Village

## Description

For the archival and development of Pocket Village, a collection of open source games set in a simple village setting, rather than a town or big city.

## More info

[https://github.com/Pocket-Village](https://github.com/Pocket-Village/)

This organization is used for the archival and development of the Pocket Village project, an open source alternative and recreation to several small village games.

## Extra info

This project is far from complete, but when ready, it will be an alternative and recreation to several small village games.
