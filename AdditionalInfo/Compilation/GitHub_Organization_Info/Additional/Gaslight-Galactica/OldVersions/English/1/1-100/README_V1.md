# GitHub organization: Gaslight-Galactica

## Description

A GitHub organization for the development of the open source Among Us recreation, Gaslight Galactica.

## More info

[https://github.com/Gaslight-Galactica](https://github.com/Gaslight-Galactica)

This organization is used for the development of the Gaslight Galactica, an open source recreation of Among Us, with new features and functionality. The name was inspired by a `Reddit:sbubby` of the Among Us logo, with the caption `Gaslight your friends`
