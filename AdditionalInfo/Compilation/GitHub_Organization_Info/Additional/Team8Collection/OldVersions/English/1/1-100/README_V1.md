# GitHub organization: Team8Collection

## Description

My GitHub organization for the development of the Team8 collection game, and references, for preserving Storm8s games.

## More info

[https://github.com/Team8Collection](https://github.com/Team8Collection)

This organization is used for the development of the Team8 game collection, an open source recreation aimed at preserving the Storm8 mobile games. This organization also serves as an extra copy of my image projects for the storm8 games I play.
