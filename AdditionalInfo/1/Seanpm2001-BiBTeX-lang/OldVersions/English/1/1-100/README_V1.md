# GitHub organization: Seanpm2001-BiBTeX-lang

## Description

For the development of my projects written in the BiBTeX language, or containing BiBTeX source code.

## More info

[https://github.com/Seanpm2001-BiBTeX-lang](https://github.com/Seanpm2001-BiBTeX-lang)
